## 日志格式说明

每条记录按日期分组，使用 Markdown 无序列表。条目格式为：

```
- **操作类型** 操作描述
```

### 操作类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **新增概念** | 在 `wiki/concepts/` 下创建了新概念笔记 | `- **新增概念** [[反向传播]] — 神经网络核心训练算法` |
| **更新概念** | 修改或扩展了已有概念笔记 | `- **更新概念** [[反向传播]] — 补充了梯度消失问题章节` |
| **新增来源** | 在 `wiki/sources/` 下创建了新来源索引 | `- **新增来源** [[Attention Is All You Need]] — Transformer 原始论文` |
| **更新来源** | 修改或扩展了已有来源索引 | `- **更新来源** [[Attention Is All You Need]] — 补充了引文地图` |
| **新建项目** | 在 `wiki/projects/` 下创建了新项目笔记 | `- **新建项目** [[图像分类实验]] — ResNet-50 在 CIFAR-100 上的实验` |
| **更新项目** | 修改或扩展了已有项目笔记 | `- **更新项目** [[图像分类实验]] — 添加了消融实验结果` |
| **初始化** | 知识库首次创建或结构初始化 | `- **初始化** 知识库初始化，wiki 结构已创建` |
| **批量导入** | 一次性从外部来源导入多条知识 | `- **批量导入** 从 Zotero 导入 12 篇论文到 inbox` |
| **链接变更** | 修复、添加或移除 wiki 页面之间的双链 | `- **链接变更** [[反向传播]] 添加了对 [[梯度下降]] 的双向链接` |
| **维护** | 格式化、标签规范化、frontmatter 修复等非内容变更 | `- **维护** 统一了 8 个概念页面的 tags 格式` |
